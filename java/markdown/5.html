解决乱码问题

乱码问题是因为编码格式不一致，我们将其改变成一致的就可以了

  1. javac程序使用UTF-8
  2. 让我们的源文件使用javac的格式也可以

我们将文件另存为utf-8格式就好了

xxxxxxxxxx
2
1
PS D:\文档\git\java\code> java Aa
2
hello word 浣犲ソ

如何改变javac的编码？

xxxxxxxxxx
1
1
javac -encoding utf-8 hello.java

 

环境搭建

IDE工具

IDE是一种集成的开发环境

  • ECLIPSE：老牌，现在用的人少

  • VScode：我主要使用的开发工具（插件丰富），可以用它来编写很多编程语言

  • IDEA：号称最智能的开发工具，虽然我没有用过，但是用的人好多

 

其他很有用的工具

Fluent Terminal

在微软商店里面即可下载

安装 WSL 命令

现在，可以在管理员 PowerShell 或 Windows 命令提示符中输入此命令，然后重启计算机来安装运行适用于 Linux 的 Windows 子系统 (WSL) 所需的全部内容。

PowerShell复制

xxxxxxxxxx
1
1
wsl --install

此命令将启用运行 WSL 并安装 Linux 的 Ubuntu 发行版所需的功能。 （可以更改此默认发行版）。

 

Snipaste 截图+贴图

Snipaste也是可以在应用商店找到，是一个特别好用的截图工具

  • 使用F1来进行截图

  • 可以固定到屏幕上
  • 可以把复制的内容变成截图F3
