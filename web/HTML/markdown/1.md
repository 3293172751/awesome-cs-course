## 字符

```
<!DOCTYPE html> 
```

**这个是文档声明标签，浏览器安装这个格式来解析代码，是HTML5的标志**



```
<html lang="en">
```

**lang：语言的缩写，告诉浏览器对应的语言是英文**

+ 中文是：`ZH`

> 这个可以对应翻译，谷歌翻译对应的语言，浏览器认为这个是英文的，那么就可以翻译给中文



```
<meta charset="UTF-8">
```

告诉浏览器我们使用的编码，这个应该和编译器的编码一致，负责就会解析错误

**一般使用`UTF-8`密码本编码，Unicode万国码字符编码**
