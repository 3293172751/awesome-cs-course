## 实现git

> 为了实施`git clone`，将涵盖以下领域：
>
> - **git 协议**用于使git 客户端能够使用各种传输机制（本机 git 协议、ssh、http）从远程 git 服务器检索所需的更改集，
> - **包文件**格式，用于将最少量的数据从服务器传输到客户端（更普遍地用于有效地将打包对象存储在存储库中），
> - git 用于以 blob 形式存储提交、树、标签和实际文件内容的底层**对象存储**
> - 以及用于跟踪工作目录中文件更改的**索引格式。**
