### rsync远程同步工具

**rsync主要用于备份和镜像，具有速度快、避免相同内容和支持符号链接的优点**

> 相比较scp,rsync速度更快，只对差异文件做更新

**第一次同步等同于拷贝**

**基本语法**

```
rsync -av atguigu@hadoop01:/opt/module atguigu@hadoop02:/opt/module
```
