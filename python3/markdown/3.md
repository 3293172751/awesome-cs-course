## 布尔值

根据条件是`True`还是 打印消息`False`：

```python
a = 200
b = 33

if b > a:
 print("b is greater than a")
else:
 print("b is not greater than a")
```

尝试

```python
In [75]: a
Out[75]: 'Hello Word'

In [76]: a == "Hello Word"
Out[76]: True

In [77]: a == 'as'
Out[77]: False
```
